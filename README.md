# TypeRoom

This is an ongoing project of a multiplayer type racing web application.

## Installation

Install using [Yarn](https://yarnpkg.com/lang/en/):

```bash
yarn install
```

or via [npm](https://www.npmjs.com/):
```bash
npm install
```

## Running

```bash
npm run dev
```

## Building

```bash
npm run build
```
